방법1.
이런식으로 되어있는데 이 pick 부분을 fixup으로 바꿔준다 두개를 바꿔준다

그럼 세개의 커밋이 하나로 된다(이건 커밋을 흡수시킴)

그러고나서 git commit --amend로 커밋메세지 바꿔주면 끝!
이걸 한꺼번에 할 수 있는게 방법2



방법2(이걸 추천함)
reflog는 했던 작업들 모두 나오고 log는 push할때 올라갈 커밋 들만나옴
그러니 log로 확인하고 HEAD~ 로 쫒아가면 됨
진짜 방법 합칠 커밋은 pick 고 바꿔줄 부분을 squash 로 바꿔주거나 약어인 s로 바꿔주면

커밋 vi 창이뜬다 여기서 커밋메세지 수정해주면 깔끔하게 하나로 통합
(바꾸려고헀던 모든 커밋들이 합쳐지며 새로운 커밋을 생성하고 이전 로그들은 싹지워짐)





reflog는 모든 작업이 다나오는거고 log는 커밋정보만 나오는거같다
